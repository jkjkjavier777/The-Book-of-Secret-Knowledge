# How to Open The Book of Secret Knowledge

## Quick start
1. Open `README.md` on GitHub to read the book-style master document.
2. Open `book_readme.html` on GitHub or download and open it in your browser for the letter-formatted layout.

## Local viewing
- Download `book_readme.html` and any referenced `assets/` files.
- Open the HTML file in Chrome, Firefox, Edge, or Safari.

## Where to find things
- `/chapters` — modular chapter files for editing.
- `/notes` — session logs and annotations.
- `render_book.py` or `render_book_weasy.py` — scripts included for maintainers (see repo for usage notes).

## Contributing
- Fork, edit chapter files, and open a pull request.
- Use issues to propose discussion questions or session assignments.
